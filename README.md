# My Discord Bot
My first bot ever made completely by me, written in Python.

## Instruction
**Bot quick guide**
- **/ai** question — Ask anything. After the answer, type **tell me more** for extra details.  
It doesn't work as an AI, but rather as a simple knowledge base. It doesn't answer everything. I considered adding AI, but it turned out to be a paid service. I've left the option of replacing it with an OpenAI key in the future.  
- **/calc** expression — Calculator (supports + - * / and quadratic equations).  
- **/plan** — Add a plan with reminders.  
- **/myplans** — View and manage your plans.  
- **/calendar** — View your monthly calendar with plan days.  
- **/quiz** — Start a quiz (up to 30 questions).  
- **/games** — Play games and earn rewards.  
- **/timer** — Focus timer with real-time countdown.  
- **/fire_lock_in** — Focus cycles with real-time countdown and stage DMs.  
- **/weather** city: — Find out the current weather and temperature in the city you specified.
